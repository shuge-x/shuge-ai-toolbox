## ADDED Requirements

### Requirement: json-formatter-tool

JSON 格式化工具提供三大功能：格式化（ beautify）、压缩（minify）、语法校验（validate）。工具页面包含输入区、操作按钮、结果展示区和错误提示区。

#### Scenario: 用户输入有效 JSON 并点击格式化按钮
- **WHEN** 用户在输入区输入有效的 JSON 文本，点击"格式化"按钮
- **THEN** 结果展示区显示格式化后的 JSON（2 空格缩进），输入区清空或保留（根据实现）

#### Scenario: 用户输入有效 JSON 并点击压缩按钮
- **WHEN** 用户在输入区输入格式化的 JSON 文本，点击"压缩"按钮
- **THEN** 结果展示区显示压缩后的 JSON（无缩进、无线），输入区清空或保留

#### Scenario: 用户输入无效 JSON 并点击格式化或压缩按钮
- **WHEN** 用户在输入区输入无效的 JSON 文本，点击"格式化"或"压缩"按钮
- **THEN** 错误提示区显示语法错误信息（包含错误描述），结果展示区不显示内容

#### Scenario: 用户输入无效 JSON 并点击语法校验按钮
- **WHEN** 用户在输入区输入无效的 JSON 文本，点击"校验"按钮
- **THEN** 错误提示区显示语法错误信息（包含错误描述和位置），结果展示区不显示内容

#### Scenario: 用户输入有效 JSON 并点击语法校验按钮
- **WHEN** 用户在输入区输入有效的 JSON 文本，点击"校验"按钮
- **THEN** 错误提示区显示"JSON 语法正确"，结果展示区显示原始 JSON

#### Scenario: 用户输入为空并点击操作按钮
- **WHEN** 用户在输入区为空（或仅包含空白字符），点击任何操作按钮
- **THEN** 错误提示区显示"请输入 JSON 数据"，结果展示区不显示内容

#### Scenario: 输入为空格或换行的 JSON
- **WHEN** 用户输入仅包含空格和换行的文本，点击格式化按钮
- **THEN** 错误提示区显示"请输入有效的 JSON 数据"

#### Scenario: 大文件 JSON 解析
- **WHEN** 用户输入一个非常大的 JSON（超过 1MB）
- **THEN** 错误提示区显示"JSON 数据过大，请控制在中。"
